## 常用命令
- 新建项目: npx @tarojs/cli init myApp
- DEV: yarn dev:weapp --watch
- 压缩启动: NODE_ENV=production taro build --type weapp --watch
- 打生产包: yarn build:weapp
- 创建文件: Taro create --name [页面名称]
- 回到某个版本: yarn global add @tarojs/cli@1.3.9
